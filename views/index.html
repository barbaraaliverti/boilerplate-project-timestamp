Timestamp Microservice

This Microservice takes a valid JavaScript date input and returns a JSON containing the unix and utc formatted date.

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000
  • [project url]/api/Dec 25 2015

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
